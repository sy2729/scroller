Scroller, a light scrolling animation library

Want a smooth scroll-reveal animation?

Try this Extremely light 1kb Javascript Scroller Library!

How To Use?

Just Include the JS File

add classname

tobeanimate

as your classname for the element you want to animate

Call function

scroller()

in your javascript file, using the default animation setting (which I already set for you)

And That's it!

If you're using a framework

You are probably familiar with lifecycle

For Example, if you're using Riot.js

Try to call scroller() after you mount your tags, like

this.on('mount', function() { scroller(); })
